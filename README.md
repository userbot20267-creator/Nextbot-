# Nextbot-
# 📚 مكتبة البوت الذكية

بوت تليجرام متكامل يعمل كمكتبة ذكية لأرشفة الكتب وتوفيرها للمستخدمين مع لوحة تحكم كاملة للمالك.

## ✨ المميزات

### 👤 للمستخدم العادي
- تصفح الأقسام والمؤلفين والكتب بنظام شجري سهل
- بحث داخلي عن الكتب (بالعنوان أو المؤلف)
- بحث خارجي تلقائي من Google Books و Open Library عند عدم وجود الكتاب
- أرشفة تلقائية للكتب القادمة من البحث الخارجي
- تحميل مباشر للكتب من داخل البوت
- نظام اشتراك إجباري للقنوات

### 🛠 للمالك (لوحة التحكم)
- إدارة الأقسام (إضافة/تعديل/حذف)
- إدارة الكتب (إضافة يدوي/تعديل/حذف)
- إدارة القنوات الإجبارية
- إدارة المستخدمين (حظر/فك حظر/إحصائيات)
- إذاعة رسائل لجميع المشتركين
- إحصائيات متقدمة (أكثر الكتب تحميلاً، أكثر الأقسام زيارة)

## 🛠 التقنيات المستخدمة

- **اللغة:** Python 3.9+
- **مكتبة تليجرام:** python-telegram-bot v20.7
- **قاعدة البيانات:** PostgreSQL
- **الاستضافة:** Render
- **البحث الخارجي:** Open Library API + Google Books API
